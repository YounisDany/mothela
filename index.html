مقدمة المشروع

فكرة المشروع

مكتبة Data Library تحتوي على جميع بيانات التقسيمات الإدارية في دولة اليمن بشكل منظم ومتسلسل، تشمل:

  • المحافظات
  • المديريات
  • العزل
  • القرى
  • التجمعات السكانية

مع الربط الكامل بين كل مستوى "Parent–Child Relationship"

المكتبة ستكون عبارة عن حزمة برمجية قابلة للاستخدام من قبل المطورين في مختلف لغات البرمجة، بهدف تسهيل الحصول على البيانات الجغرافية والإدارية الموثوقة دون الحاجة لبناء قواعد بيانات خاصة.

أمثلة استخدام

JavaScript (npm)
npm install yemen-regions

import { getGovernorates, getDistricts } from 'yemen-regions';

// جلب جميع المحافظات
const governorates = getGovernorates();

// جلب المديريات التابعة لمحافظة معينة
const districts = getDistricts('صنعاء');
Python (pip)
pip install yemen_regions

from yemen_regions import YemenRegions

yr = YemenRegions()

# جلب جميع المحافظات
governorates = yr.get_governorates()

# جلب المديريات التابعة لمحافظة معينة
districts = yr.get_districts('عدن')
استخدام CDN
// جلب بيانات المحافظات عبر CDN
fetch('https://cdn.yemen-regions.com/governorates.json')
  .then(response => response.json())
  .then(data => {
    console.log(data);
    // استخدام البيانات
  });
